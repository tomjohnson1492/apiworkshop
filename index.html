This is a simple index file that appears when you go to the homepage of the API Workshop repository (https://github.com/tomjohnson1492/apiworkshop). I set up Github Pages with this repository, so you can view any file in the repository by simply adding the correct path in the URL.

For example, to view the eventbrite.html file in the repository, go to https://github.com/tomjohnson1492/apiworkshop/eventbrite.html. That's it.
